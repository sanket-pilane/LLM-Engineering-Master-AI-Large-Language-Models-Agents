# 🤖 LLM Engineering – Master AI, Large Language Models & Agents

## 📌 Overview
This repository contains my learning and practical implementations from the **LLM Engineering** course. The focus of this course is to understand how Large Language Models (LLMs) work and how to build real-world AI applications using modern tools and frameworks.

### 🔹 LLM Fundamentals

- Tokens, context window, and transformer basics
- Attention mechanism (high-level)

---

### 🔹 Prompt Engineering
- Writing effective prompts
- Few-shot and chain-of-thought prompting
- Improving response quality

---

### 🔹 Retrieval-Augmented Generation (RAG)
- Combining LLMs with external data
- Embeddings and vector databases
- Document-based question answering systems

---

### 🔹 LangChain
- Chains and workflows
- Agents and tool usage
- Memory integration

### 🔹 Hugging Face

- Model loading and inference

---

### 🔹 AI Agents
- Tool calling
- Autonomous task execution
- Multi-agent systems

---

### 🔹 Fine-Tuning (Basics)

### 🔹 Building AI Applications

- API-integrated AI systems
